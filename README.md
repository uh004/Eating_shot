## EatingShot : 당뇨병 환자들을 위한 식단 조절 및 케어 서비스

> 개발 기간 
- 2024.07.01 ~ 2024.11.18
<br>

> Stack
- **AI** : Python, Ultralytics
- **FrontEnd** :  HTML, CSS, JavaScript
- **BackEnd** : Django, FastAPI 
- **Database** : SQLite 
- **Tools & Services** : Docker, Redis, Celery
<br>
